## INSTALLATION

```bash
sudo apt install python3.7
sudo apt install python3-pip
apt-get install tesseract-ocr libtesseract-dev libleptonica-dev pkg-config
pip3 install tesserocr
```

## EXECUTION

```bash
python3 src/main.py
```
